Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai retenu que l’environnement web repose sur une architecture client-serveur, où le navigateur envoie des requêtes au serveur pour afficher des pages web. J’ai appris les bases du HTML pour structurer le contenu, du CSS pour le styliser, et du JavaScript pour le rendre interactif. J’ai aussi compris le rôle des protocoles comme HTTP et HTTPS, ainsi que la notion de serveur local pour tester nos projets en développement.

Pratiques (quels sont les logiciels que vous avez appris)?

J’ai appris à utiliser Visual Studio Code comme éditeur de code principal. J’ai découvert comment installer et configurer un serveur local avec XAMPP ou Live Server. J’ai utilisé Git et GitHub pour le versionnage et la collaboration. Enfin, j’ai exploré les outils de développement intégrés dans les navigateurs (comme les DevTools de Chrome) pour inspecter et corriger le code.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai réalisé une page web statique en HTML et CSS pour présenter un portfolio. J’ai également développé une maquette interactive avec du JavaScript pour simuler une navigation dynamique. Un autre projet consistait à créer une page de contact avec formulaire, hébergée sur un serveur local. Ces projets m’ont permis de mieux comprendre le fonctionnement d’un environnement web et de mettre en pratique les notions apprises.
