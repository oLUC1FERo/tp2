Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai retenu que la programmation web repose sur trois langages principaux : HTML pour structurer le contenu, CSS pour le styliser, et JavaScript pour le rendre interactif. J’ai appris les bases du fonctionnement d’une page web, le rôle du navigateur, et la manière dont les fichiers sont interprétés. J’ai aussi compris l’importance de la syntaxe, de la hiérarchie des balises, et de la séparation entre le contenu et la présentation.

Pratiques (quels sont les logiciels que vous avez appris)?

J’ai appris à utiliser Visual Studio Code comme éditeur de code. J’ai découvert les outils de développement intégrés dans les navigateurs pour inspecter et corriger le code. J’ai également utilisé Live Server pour visualiser mes pages en temps réel, et GitHub pour sauvegarder et partager mes projets. Ces outils m’ont permis de travailler plus efficacement et de mieux comprendre le cycle de développement.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai réalisé une page de présentation personnelle en HTML et CSS, avec une mise en page simple et responsive. J’ai aussi créé un formulaire de contact fonctionnel, et une petite galerie d’images. Ces projets m’ont permis de mettre en pratique les bases de la programmation web et de développer une première approche structurée du code.
