Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai retenu que le design web repose sur des principes fondamentaux tels que l’ergonomie, la hiérarchie visuelle, la lisibilité et l’accessibilité. La théorie des couleurs, la typographie, et la disposition des éléments sont essentielles pour créer une interface cohérente et agréable à utiliser. J’ai également appris à respecter les normes du responsive design afin d’assurer une expérience fluide sur tous les types d’appareils.

Pratiques (quels sont les logiciels que vous avez appris)?

J’ai appris à utiliser Figma pour concevoir des maquettes interactives et collaborer avec les autres membres de l’équipe. J’ai également utilisé Canva pour créer des visuels simples et efficaces, ainsi que Adobe Photoshop pour la retouche d’images. Visual Studio Code m’a permis d’intégrer les éléments graphiques dans le code HTML et CSS tout en respectant les maquettes validées.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai participé à la création de l’identité visuelle d’un site vitrine, en définissant une charte graphique cohérente. J’ai conçu des maquettes pour plusieurs pages du site, en tenant compte de l’expérience utilisateur. J’ai également réalisé des icônes personnalisées et des visuels adaptés au contenu. Enfin, j’ai intégré le design dans le code en respectant les contraintes techniques du responsive design.
