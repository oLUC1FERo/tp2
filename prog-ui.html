Communication au sein d’une équipe de projet Web

Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai retenu que la communication est un élément central dans le bon fonctionnement d’une équipe. Les modèles théoriques comme celui de Shannon et Weaver nous aident à comprendre comment les messages circulent et comment les interférences peuvent nuire à leur réception. J’ai aussi appris l’importance de la communication non verbale, de l’écoute active et de la gestion des conflits pour maintenir une ambiance de travail productive et respectueuse.

Pratiques (quels sont les logiciels que vous avez appris)?

J’ai appris à utiliser plusieurs logiciels essentiels pour le travail en équipe sur un projet web. Visual Studio Code m’a permis de coder efficacement. GitHub a été utilisé pour le versionnage et la collaboration. Figma m’a servi à créer des maquettes visuelles du site. Trello m’a aidé à organiser les tâches, et Slack a facilité les échanges rapides entre les membres du groupe.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai participé à la création d’un site vitrine pour une entreprise fictive, avec un design responsive et une navigation intuitive. J’ai également contribué à la mise en place d’un mini-CMS permettant de gérer du contenu facilement. Enfin, nous avons développé un blog collaboratif avec système de commentaires, ce qui m’a permis de mieux comprendre les aspects dynamiques du développement web.
