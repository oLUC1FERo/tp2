Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai retenu que la programmation d’interface web repose sur la structuration du contenu avec HTML, la mise en forme avec CSS, et l’interactivité avec JavaScript. J’ai appris les principes de base du DOM (Document Object Model), qui permet de manipuler dynamiquement les éléments d’une page. J’ai aussi compris l’importance de l’accessibilité, de la compatibilité entre navigateurs, et de la performance dans la conception d’interfaces efficaces.

Pratiques (quels sont les logiciels que vous avez appris)?

J’ai appris à utiliser Visual Studio Code pour coder mes interfaces. J’ai utilisé les outils de développement intégrés dans les navigateurs pour inspecter et corriger le code. J’ai aussi découvert des bibliothèques comme jQuery pour simplifier certaines interactions, et j’ai commencé à explorer des frameworks CSS comme Bootstrap pour accélérer la mise en page. GitHub m’a permis de versionner mon travail et de collaborer avec mes collègues.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai réalisé une interface de formulaire avec validation JavaScript, une galerie d’images interactive, et une page d’accueil avec menu déroulant et effets visuels. Ces projets m’ont permis de mettre en pratique les notions de programmation front-end et de mieux comprendre comment structurer une interface intuitive et responsive.
