Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai retenu que la communication est essentielle pour assurer la cohésion et l’efficacité d’une équipe. Les modèles de communication comme celui de Shannon et Weaver permettent de comprendre les étapes de transmission d’un message et les obstacles possibles. Les notions de communication interpersonnelle, de gestion des conflits et de dynamique de groupe sont également fondamentales pour favoriser un environnement de travail sain et productif.

Pratiques (quels sont les logiciels que vous avez appris)?

J’ai appris à utiliser plusieurs logiciels utiles dans le cadre d’un projet web. Visual Studio Code m’a permis de coder efficacement en HTML et CSS. GitHub a été essentiel pour le partage de fichiers et le suivi des versions. Figma m’a servi à concevoir des maquettes interactives, Trello à organiser les tâches, et Slack à communiquer rapidement avec les membres de l’équipe.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai participé à la création d’un site vitrine pour une entreprise fictive, avec un design responsive et une navigation intuitive. J’ai également contribué à la mise en place d’un mini-CMS permettant de gérer du contenu facilement. Enfin, nous avons développé un blog collaboratif avec système de commentaires, ce qui m’a permis de mieux comprendre les aspects dynamiques du développement web.
