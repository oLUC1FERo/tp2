Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Nous avons retenu plusieurs notions essentielles liées à la communication en équipe. Parmi elles, le modèle émetteur-récepteur, les types de communication (verbale, non verbale, écrite), et les dynamiques de groupe. La théorie de Tuckman sur les étapes de formation d’une équipe (formation, agitation, normalisation, performance) nous a permis de mieux comprendre les interactions entre les membres. Ces concepts nous ont aidés à structurer nos échanges et à favoriser un climat de collaboration.

Pratiques (quels sont les logiciels que vous avez appris)?

Nous avons utilisé plusieurs logiciels pour faciliter notre travail en équipe. Visual Studio Code a été notre principal éditeur de code. GitHub nous a permis de gérer les versions et de collaborer efficacement. Figma a servi à concevoir les maquettes du site, tandis que Trello nous a aidés à organiser les tâches. Slack a été utilisé pour la communication rapide entre les membres, assurant une coordination fluide tout au long du projet.

Conditionnelles (quels sont les projets que vous avez réalisés)?

Au cours du projet, nous avons réalisé plusieurs livrables concrets. Nous avons développé un site vitrine pour une entreprise fictive, avec un design responsive et une navigation intuitive. Un mini-CMS a été mis en place pour permettre la gestion de contenu. Nous avons également conçu un blog collaboratif avec un système de commentaires. Ces projets nous ont permis de mettre en pratique nos compétences techniques et de renforcer notre esprit d’équipe.
