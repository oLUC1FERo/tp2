Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai approfondi mes connaissances sur les interactions entre le front-end et le back-end. J’ai compris comment les requêtes HTTP sont traitées par le serveur, et comment les bases de données sont utilisées pour stocker et récupérer des informations. J’ai également appris les principes de sécurité web comme la gestion des sessions, la validation des formulaires, et la protection contre les injections SQL. Ces notions sont essentielles pour construire des applications web robustes et sécurisées.

Pratiques (quels sont les logiciels que vous avez appris)?

J’ai appris à utiliser des environnements de développement plus complets comme XAMPP pour simuler un serveur local avec PHP et MySQL. J’ai utilisé phpMyAdmin pour gérer les bases de données. J’ai également exploré Postman pour tester les API, et j’ai continué à utiliser Visual Studio Code pour coder en HTML, CSS, JavaScript et PHP. GitHub est resté notre outil de versionnage et de collaboration.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai réalisé un site dynamique avec formulaire de contact connecté à une base de données. J’ai aussi développé une interface d’administration permettant d’ajouter, modifier et supprimer du contenu. Un autre projet consistait à créer une petite application web avec authentification utilisateur, ce qui m’a permis de mettre en pratique la gestion des sessions et la sécurisation des données.
