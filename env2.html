Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

J’ai retenu que la communication est un pilier fondamental dans le bon déroulement d’un projet web. Les notions de communication interpersonnelle, de gestion des conflits et de coordination sont essentielles pour maintenir une dynamique de groupe efficace. Les modèles théoriques comme celui de Shannon et Weaver permettent de mieux comprendre le processus de transmission d’un message, en identifiant les sources de bruit et les obstacles à la compréhension.

Pratiques (quels sont les logiciels que vous avez appris)?

Durant ce projet, j’ai appris à utiliser plusieurs outils collaboratifs et techniques. Visual Studio Code m’a permis de coder efficacement en HTML et CSS. J’ai découvert GitHub pour le versionnage et la collaboration en ligne. Figma m’a servi à concevoir des maquettes interactives, tandis que Trello m’a aidé à organiser les tâches. Slack a été utilisé pour échanger rapidement avec les membres de l’équipe.

Conditionnelles (quels sont les projets que vous avez réalisés)?

J’ai participé à la réalisation d’un site vitrine pour une entreprise fictive, avec un design responsive et une navigation intuitive. J’ai également contribué à la création d’un mini-CMS permettant la gestion de contenu via une interface simple. Enfin, nous avons mis en place un blog collaboratif intégrant un système de commentaires, ce qui m’a permis de mieux comprendre les enjeux du développement dynamique.
